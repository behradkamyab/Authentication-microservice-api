# Authentication Service
## Backend API with mongodb, nodejs, expressj, jwt

This Authentication service work with phone number and password and it has:
1. Confirmation Code functionality after signing up
2. Password has the most complexity for validating
3. Reset Password functionality
4. You can enable or disbale OTP functionality for a user and if it is on after validating credentials, otp code will be send to your phone number


## Features

- Password complexity
- JWT & Crypto for tokens
- password reseting (for now without sending the link email)
- OTP
- Disbale or enable OTP
- error handling with next middleware
- Restful
- timestamps


## Tech

Dillinger uses a number of open source projects to work properly:

- Nodejs for the backend
- Express - fast node.js network app framework [@tjholowaychuk]
- Mongodb - atlas for db
- JWT - tokens
